# Бот конкурентной аналитики коммерческой недвижимости

Telegram-бот для мониторинга конкурентов бизнес-центра **«Ельцовка-1»**. Проект собирает данные по свободным помещениям, хранит историю изменений, поддерживает ручной ввод по объектам без сайта и формирует Excel-отчеты для руководства.

## Что умеет
- парсит конкурентов с сайтом (сейчас подключен **Сибирский бизнес парк / SibBP**);
- ведет ручной учет конкурентов без сайта;
- хранит жизненный цикл объектов: `first_seen`, `last_seen`, `active / unconfirmed / removed`;
- считает количество помещений, суммарную площадь, средневзвешенную ставку и общую стоимость;
- показывает изменения, архив и приоритет прозвона;
- формирует Excel-отчеты по конкуренту и по всему портфелю;
- строит динамику свободных площадей и количества объектов;
- автоматически отправляет сводку по расписанию в Telegram.

## Бизнес-ценность
- сокращает время на ручной мониторинг конкурентов;
- дает единый формат отчетности по рынку;
- помогает отслеживать динамику свободных площадей;
- создает основу для принятия решений по арендным ставкам и позиционированию.

## Автоматическое расписание
- **Понедельник и четверг, 10:00 МСК** — общая сводка + Excel
- **Понедельник, 10:20 МСК** — Excel с динамикой
- **Понедельник, 09:40 МСК** — напоминание по ручной верификации / прозвону

## Как запустить локально
1. Создай виртуальное окружение.
2. Установи зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Создай файл `.env` или задай переменные окружения:
   ```env
   BOT_TOKEN=your_telegram_bot_token
   CHAT_ID=your_telegram_chat_id
   BASE_URL=https://sibbp.ru/
   REQUEST_TIMEOUT=30
   ```
4. Запусти бота:
   ```bash
   python main.py
   ```

## Деплой на Render
1. Создай новый GitHub-репозиторий и загрузи туда этот проект.
2. В Render создай **Background Worker** или используй `render.yaml`.
3. Добавь переменные окружения:
   - `BOT_TOKEN`
   - `CHAT_ID`
   - `BASE_URL`
   - `REQUEST_TIMEOUT`
4. Запусти деплой.

## Что важно перед публикацией
В исходном архиве токен Telegram был сохранен прямо в `config.py`. Для публичного репозитория это небезопасно. В этой версии проекта токен убран, а настройки читаются из переменных окружения.
